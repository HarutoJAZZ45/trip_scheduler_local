# 旅のしおりアプリ (Ryoko no Shiori)

ポルトガル・スペイン旅行（202x年2月27日〜3月11日）のための、iPhone向け旅のしおりWebアプリケーションです。
スタイリッシュなデザインと、オフラインでも使えるような手軽な操作性を目指して開発されました。

## 📱 プロジェクト概要

*   **ターゲット**: iPhone (モバイルファーストデザイン)
*   **デザイン**: スペイン・ポルトガルの雰囲気を意識したモダンなUI (テラコッタ、アズレージョブルー)
*   **主な機能**: 全データはブラウザの `localStorage` に保存され、サーバー不要でデータの編集・永続化が可能です。

## 🚀 機能一覧

### 1. ホーム (Home)
*   旅行の概要（日付、都市数）を表示。
*   没入感のあるカバー画像とアニメーション。

### 2. 行程表 (Schedule)
*   **タイムライン表示**: フライト、移動、観光、食事などのイベントを時系列で確認可能。
*   **リアルデータ**: イスタンブール経由のターキッシュエアラインズ等のフライト情報をプリセット。
*   **編集機能**: 右下の「＋」ボタンから予定の追加が可能。削除も可能。

### 3. 準備リスト (Packing)
*   **持ち物リスト**: 「必需品」「ガジェット」「衣類」などのカテゴリー分け。チェックボックスの状態は保存されます。
*   **予算管理**: 航空券やホテル代などの概算を表示。
*   **編集機能**: 独自の持ち物を追加可能。

### 4. スポット (Spots)
*   **行きたい場所**: 気になるカフェや観光地をカード形式で保存。
*   **編集機能**: 新しいスポットを写真（プレースホルダー）付きで追加可能。

## 🛠 技術スタック

*   **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animation**: [Framer Motion](https://www.framer.com/motion/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **State Management**: Custom `useLocalStorage` Hook (SSR/Hydration safe)

## 💻 開発・実行方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
# http://localhost:3000 でアクセス
```

## ⚠️ 移行時の注意点
このアプリは `localStorage` を使用しているため、データは「ブラウザごと」に保存されます。
PCを変えたり、ブラウザを変えたりすると、追加・編集したデータは初期状態（プリセットデータ）に戻ります。
（ソースコード上の `INITIAL_SCHEDULE` などの定数が初期値として使われます）
